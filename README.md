# 📖 Cryptopedia <a name="about-project"></a>

 NEWS HUB is a Web-App for real-time news articles around the World. Users can see latest news and can also filter news based on sources. NEWS HUB consumes displays articles provided by [NEWSORG](http://newsapi.org/) API.

## 💻 Getting Started <a name="getting-started"></a>

> 
To get a local copy up and running, follow these steps.

- npm
  ```sh
  npm install npm@latest -g
  ```

### Setup

1. Clone the repo by running the command
   ```sh
    git clone https://github.com/V-Blaze/news-hub.git
   ```
2. Open the directory of the project
   ```sh
   cd news-hub
   ```
3. Install project dependencies
   ```sh
   run npm install
   ```
4. Create a `.env` file in the root directory and add the following content
   ```sh
   REACT_APP_BASE_URL=https://newsapi.org/v2/
   REACT_APP_API_KEY=15e37ec6494740dc83c661dd41d2b516
   ```
5. Open the html file
   ```sh
   run npm start
